The Challenge

Femur fractures require rapid, accurate diagnosis

Femur fractures are among the most critical orthopedic emergencies. Delayed or missed diagnoses can lead to severe complications. FemurUI leverages deep learning to provide instant, reliable fracture detection with explainable AI insights.

Capabilities

Key Features

Deep Learning Classification
ResNet50V2 with transfer learning provides 93.27% accuracy for binary fracture detection on femur X-rays.
Saliency Map Visualization
Gradient-based heatmap shows exactly which regions the AI model focuses on, with bounding box highlighting.
GPT-4 Expert Explanation
AI-generated radiological assessments describe fracture type, location, significance, and treatment options.
Modern Desktop UI
PyQt6 with QML provides a professional, fluid interface with smooth animations and responsive layout.
Clinical Safety Warnings
Built-in disclaimers ensure the tool is used as decision support, always recommending professional consultation.
Real-time Analysis
Instant inference with progress indication and seamless result rendering upon completion.
